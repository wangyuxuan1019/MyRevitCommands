# MyRevitCommands
.NET C# based plugins for Revit on both interface and database level inluding :

    GetElementId - Get the Id from an element
    
    CollectWindows - Collect all the windows from a document
    
    DeleteElement - Delete an element from a document
    
    PlaceFamily - Place a family(chair)
    
    PlaceLineElement - Place a line element(wall)
    
    PlaceLoopElement - Place a loop element(floor)
    
    GetParameter - Get a parameter(window height)
    
    SetParameter - Set a parameter(eg. window height)
    
    ChangeLocation - Change the location of an element(eg. shift East 3')
    
    Edit element - Edit an element(eg. shit NorthEast 3' and rotate 30 degree)
    
    SelectGeometry - Select geometry and calculate the area of all the faces
    
    ElementIntersection - Select geometries that have intersection
    
    CreatePlanView - Create a plan view(eg. level 1)
    
    ViewFilter - Filter section views(eg. viewname withOUT "WIP")
    
    TagView - Tag elements in a view(eg. windows)
    
    Sheets - create a sheet with titleblock
    
    PlaceView - Place a view on a sheet
    
    Application & DBApplication
